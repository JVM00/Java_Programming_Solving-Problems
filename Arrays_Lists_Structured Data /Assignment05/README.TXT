PROJECT TITLE: Assignment05 - GladLib Story Generator
PURPOSE OF PROJECT: Generate randomized mad-lib style stories by replacing template placeholders with words loaded from category files.
VERSION or DATE: 2026-05-08
HOW TO START THIS PROJECT: Open the folder in BlueJ, create a GladLib object, and run makeStory(). The default constructor uses the local data folder.
AUTHORS: JVM
USER INSTRUCTIONS: The data and datalong folders contain word lists and templates. Placeholders such as <noun>, <adjective>, <color>, <country>, <name>, <animal>, <timeframe>, <verb>, <fruit>, and <number> are replaced when the story is generated. The program avoids reusing the same substitute word in one story and prints the number of replacements made.
