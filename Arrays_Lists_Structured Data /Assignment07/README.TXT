PROJECT TITLE: Assignment07 - Web Log Analyzer
PURPOSE OF PROJECT: Parse web server log files and analyze IP addresses, status codes, visit counts, and daily traffic patterns using ArrayList and HashMap collections.
VERSION or DATE: 2026-05-08
HOW TO START THIS PROJECT: Open the folder in BlueJ. Create a Tester object and run the test methods for the analysis you want, such as testUniqueIP(), testStatusCodesHigherThan400(), testCountVisitsPerIP(), or testDayWithMostIPVisits().
AUTHORS: JVM 
USER INSTRUCTIONS: LogAnalyzer reads log files with readFile(filename), stores LogEntry records, and provides methods for counting unique IPs, filtering status codes, finding visits on a day, counting visits per IP, grouping IPs by day, and finding the busiest day or IPs. Sample logs include short-test_log, weblog-short_log, weblog1_log, weblog2_log, and weblog3-short_log.
