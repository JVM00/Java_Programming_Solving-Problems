PROJECT TITLE: Assignment02 - Caesar Breaker and Word Lengths
PURPOSE OF PROJECT: Analyze text with arrays and frequency counts, including word-length statistics and Caesar cipher decryption for one-key and two-key encrypted messages.
VERSION or DATE: 2026-05-08
HOW TO START THIS PROJECT: Open the folder in BlueJ. Run WordLengths.testCountWordLengths() to choose a text file and count word lengths, or create a CaesarBreaker object and run testDecrypt(), testHalfOfString(), testDecryptTwoKeys(), or decryptTwoKeys_file().
AUTHORS: JVM 
USER INSTRUCTIONS: WordLengths requires the edu.duke FileResource chooser. CaesarBreaker uses letter frequency and a small scoring pass to recover Caesar keys. Sample files include lotsOfWords.txt, romeo.txt, smallHamlet.txt, mysteryTwoKeysPractice.txt, and decrypted_two_keys_output.txt.
