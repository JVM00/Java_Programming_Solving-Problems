PROJECT TITLE: Assignment06 - Codon Count, Words in Files, and GladLibMap
PURPOSE OF PROJECT: Count DNA codons for each reading frame, determine which words appear in the most selected files, and generate GladLib stories using a map of category words.
VERSION or DATE: 2026-04-07
HOW TO START THIS PROJECT: Open the project in BlueJ and run the tester() method in CodonCount or WordsInFiles, or run makeStory() in GladLibMap.
AUTHORS: JVM with Codex assistance
USER INSTRUCTIONS: For CodonCount, choose a DNA text file when prompted. For WordsInFiles, choose a group of text files when prompted. For GladLibMap, run makeStory() and it will use the local data folder and template files. Sample files for all three tasks are included in this project.
