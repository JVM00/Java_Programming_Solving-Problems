PROJECT TITLE: Assignment06 - Codon Count, Words in Files, and GladLibMap
PURPOSE OF PROJECT: Count DNA codons for each reading frame, determine which words appear in the most selected files, and generate GladLib stories using a map of category words.
VERSION or DATE: 2026-05-08
HOW TO START THIS PROJECT: Open the project in BlueJ and run the tester() method in CodonCount or WordsInFiles, or run makeStory() in GladLibMap.
AUTHORS: JVM 
USER INSTRUCTIONS: For CodonCount, choose a DNA text file when prompted. It reports unique codons, the most common codon, and codons in a requested count range for reading frames 0, 1, and 2. For WordsInFiles, choose a group of text files when prompted; it maps each lowercase word to the files containing it and can report words found in a chosen number of files. For GladLibMap, run makeStory() and it will use the local data folder and template files. Sample DNA, brief text files, and GladLib data files are included in this project.
