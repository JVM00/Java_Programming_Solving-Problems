PROJECT TITLE: Assignment01 - WordPlay and Caesar Cipher
PURPOSE OF PROJECT: Practice string traversal, character testing, vowel replacement, character emphasis, and Caesar cipher encryption with one or two keys.
VERSION or DATE: 2026-05-08
HOW TO START THIS PROJECT: Open the folder in BlueJ and create a WordPlay or CaesarCipher object. Run WordPlay.test(), CaesarCipher.testEncrypt(), or CaesarCipher.testEncryptTwoKeys(). The main methods can also be run from a Java IDE.
AUTHORS: JVM
USER INSTRUCTIONS: WordPlay demonstrates isVowel, replaceVowels, and emphasize on sample strings. CaesarCipher encrypts alphabetic characters while preserving case and leaving punctuation unchanged. To encrypt file contents, run testCaesar() in BlueJ/Coursera with the edu.duke library loaded and choose a text file such as sample_testCaesar.txt.
