# Java Programming: Solving Problems

This repository contains Java practice projects and course assignments focused on strings, files, arrays, lists, maps, parsing data, debugging, and simple encryption/decryption programs.

Most projects come from Duke/Coursera-style Java programming exercises and are organized as either BlueJ projects or small Eclipse/VS-style Java folders.

## Repository Structure

- `Arrays_Lists_Structured Data /` - eight BlueJ assignments covering Caesar ciphers, word counts, ArrayList and HashMap practice, GladLib story generation, web log analysis, and Vigenere cipher breaking.
- `Solving_Problems_With_Software/` - earlier Java exercises for strings, parsing CSV/export data, parsing weather data, perimeter calculations, baby-name data, debugging, and hello-world projects.
- `Debugging/`, `ParsingExportData/`, `StringFirstAssignments/`, `StringsSecondAssignments/`, `StringsThirdAssignments/`, `hello_VS/`, and `hello_eclipse/` - standalone or IDE-generated project folders.
- `apache-csv.jar` and `edu.duke.jar` - supporting libraries used by several assignments.

## Requirements

- Java JDK
- BlueJ for the BlueJ assignment folders
- `edu.duke.jar` for Duke FileResource, DirectoryResource, URLResource, CSVParser, and related helper classes
- `apache-csv.jar` for CSV parsing assignments

## How To Run

For BlueJ projects:

1. Open the assignment folder that contains `package.bluej`.
2. Compile the project in BlueJ.
3. Create the tester or main class object listed in that assignment's `README.TXT`.
4. Run the relevant test method.

For standard Java IDE projects:

1. Open the folder in Eclipse, VS Code, or another Java IDE.
2. Add `edu.duke.jar` and `apache-csv.jar` to the classpath when the code imports `edu.duke` or Apache CSV classes.
3. Run the class that contains `main`, or run the public test methods from the IDE.

## Assignment README Files

The `Arrays_Lists_Structured Data /Assignment01` through `Assignment08` folders each include a completed `README.TXT` with the project title, purpose, author note, and user instructions for that assignment.

## Notes

Some folders include `.class`, `.ctxt`, `bin`, `.metadata`, and IDE project files generated by BlueJ or Eclipse. Source code is in the `.java` files, and sample input data is stored beside the assignments that use it.
